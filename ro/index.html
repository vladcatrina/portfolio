Repozitoriu

GitHub Pages

Este o aplicație single-page în care îți poți gestiona treburile de zi cu zi. Sarcinile pot fi de două feluri: Quest și Training. Quest-urile te recompensează cu experiență, în timp ce Training-urile îți măresc nivelul unei anume abilități. Quest-ul poate fi condiționat și de atingerea unui anume nivel de abilitate. Ambele tipuri de sarcini pot fi contratimp, iar aplicația are un dropdown cu notificări, unde poți vedea sarcinile expirate și pe cele care au intrat în ultimul sfert al timpului alocat.

Aplicația a fost realizată cu framework-ul Svelte. Nu are back-end - datele sunt reținute în local storage. Nu am folosit alte dependințe de JS sau CSS.

Repozitoriu

Este un site web în care poți viziona date fotbalistice obținute prin API-ul Football-Data.org. Datele sunt primite în format JSON și transformate de o aplicație node.js cu express.js în template-uri HTML cu ajutorul sistemului de templating EJS. Componentele de tip tabel sunt generate printr-o singură funcție, care permite derivarea câmpurilor și paginarea; aceasta returnează atât tabelul HTML cât și un cod Javascript pentru paginare, ambele prin magia EJS.

Fiecare competiție, echipă, meci sau jucător poate avea propriul fir de discuție pe forum. Aceste fire pot conține comentarii arborescente (recursive). Firele de discuție sunt reținute într-o colecție MongoDB aflată în cloud.

În acest joc platformer, un savant țicnit încearcă să scape din propriul laborator, însă laboratorul e infinit, generând automat câte un nou nivel dintr-o listă predefinită de scheme JSON. Pentru a evita obstacolele și armele laser automate, savantul are un as în mânecă - poate activa starea de imponderabilitate (Zero G).

Jocul a fost realizat în TypeScript cu engine-ul Phaser.js. Sunetele folosite sunt enumerate în readme-ul repozitoriului. Grafica este creată tot de mine, mai puțin trupul personajului, care este o variantă recolorată a trupului omulețului din tutorialul Phaser, și câteva elemente din meniu.

Problemele de framerate din videoul de mai jos nu se găsesc în jocul real. Poți verifica asta urmând link-ul GitHub Pages. ;)
